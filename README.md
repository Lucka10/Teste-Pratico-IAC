# Seleção para Auxiliar de Programação - Teste Prático - IAC

## Agenda de Contatos

### Frontend
- dependencias
    - Vue.js
    - Bootstraps CDN
    - Typescript
    - axios
    - input-mask

Desenvolvimento:
```sh
npm run dev
```
Testes:
```sh
npm run test:unit
```

Produção:
```sh
npm run build
npm run start
```

### backend
- frameworks
    - Django

iniciar servidor
```sh
python manage.py runserver
```
